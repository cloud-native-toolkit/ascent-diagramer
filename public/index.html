Ascent Diagramer

Python app exposing REST APIs for building diagrams out of Bill of Materials definition.

This Flask microservice has these endpoints:
  • /health - returns a simple {"status":"UP"} json response if the app is running
  • /diagram - returns an infrastructure diagram image built from uploaded bom.yaml bill of materials definition
  • /software - returns a software diagram image built from uploaded bom.yaml bill of materials definition
  • /swagger/api-docs - standard Swagger UI to exercise all the endpoints
